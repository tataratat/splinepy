Detailed Description

Similar to ControlPointPointers, but for weights. This same layout as ControlPointPointers, because in BSplineLib, weights are saved as last element of the homogeneous control points, which means that it is not contiguous. bezman, however, has a contiguous array, but this approach is applicable for both case.
